# Dynamic Demo Next

Next.js demo app for `@dynamic-field-kit/react`.

## What It Does

- Registers local field renderers from the app registry file
- Renders a dynamic form in `app/page.tsx`
- Shows live JSON output for the current values

## Run

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Build

```bash
npm run build
npm run start
```

## Main Files

- `app/page.tsx`
- `app/globals.css`
- `postcss.config.mjs`
- `next.config.ts`
