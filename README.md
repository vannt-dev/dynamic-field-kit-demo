# dynamic-field-kit-demo

Demo apps for `Dynamic Field Kit` across multiple frameworks.

## Apps

- `react-app`: Next.js + React demo using `@dynamic-field-kit/react`
- `dynamic-demo-next`: another Next.js + React demo
- `dynamic-demo-vite`: Vite + React demo
- `vue-app`: Vue 3 + Vite demo using `@dynamic-field-kit/vue`
- `angular-app`: Angular demo using `@dynamic-field-kit/angular`

Each app renders a simple dynamic form and shows the current form state as JSON.

## Run

Install dependencies and run each app from its own folder.

React Next.js demo:

```bash
cd react-app
npm install
npm run dev
```

Next.js demo:

```bash
cd dynamic-demo-next
npm install
npm run dev
```

React Vite demo:

```bash
cd dynamic-demo-vite
npm install
npm run dev
```

Vue demo:

```bash
cd vue-app
npm install
npm run dev
```

Angular demo:

```bash
cd angular-app
npm install
npm start
```
