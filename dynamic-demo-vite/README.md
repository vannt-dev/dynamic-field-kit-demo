# Dynamic Demo Vite

Vite + React demo app for `@dynamic-field-kit/react`.

## What It Does

- Registers field renderers locally
- Renders a dynamic form in `src/App.tsx`
- Shows live JSON output for form state

## Run

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Main Files

- `src/App.tsx`
- `src/fieldRegistry.ts`
- `src/form.ts`
